# Libft

## Functions :

**ft_atoi.c** -> Converts a string to an integer.

**ft_bzero.c** -> Sets a block of memory to zero.

**ft_calloc.c** -> Allocates memory and initializes it to zero.

**ft_contain_char.c** -> check if the str contain the char : 1 -> it does, 0 -> it does not.

**ft_dba_dup.c** -> Duplicates a double array and return an allocated copie.

**ft_dba_len.c** -> Return the nb of string in a double array, -1 if NULL.

**ft_extract_path** -> return the line after the '=' in a envp variable.

**ft_free_before_i** -> free all allocated string before i in a double arra (in case of error).

**ft_free_double_tab** -> to free all element of a **tab + the tab.

**ft_isalnum.c** -> check if the char is a alphanumerical : 1 -> it is, 0 -> it is not.

**ft_isalpha.c** -> check if the char is a letter : 1 -> it is, 0 -> it is not.

**ft_isascii.c** -> check if the char is in the ASCII table : 1 -> it is, 0 -> it is not.

**ft_isdigit.c** -> check if the char is a digit : 1 -> it is, 0 -> it is not.

**ft_isprint.c** -> check if the char is a printable : 1 -> it is, 0 -> it is not.

**ft_itoa.c** -> Converts an integer to a string -> return that allocated string.

**ft_memchr.c** -> Searches for a byte in a block of memory.

**ft_memcmp.c** -> Compares two blocks of memory.

**ft_memcpy.c** -> Copies a block of memory from one location to another.

**ft_memmove.c** -> Copies a block of memory safely, even if the memory areas overlap.

**ft_memset.c** -> Filles a block of memory with a specific byte value. (init to 0);

**ft_putchar_fd.c** -> Write a char to a chosen fd.

**ft_putendl_fd.c** -> Write a string followed by a newline to a chosen fd.

**ft_putnbr_fd.c**-> Write a digit to a chosen fd.

**ft_putstr_fd.c** -> Write a string to a chosen fd.

**ft_split.c** -> Splits a string into an array of strings using a (char) delimiters -> return the allocated **tab containing all the allocated string.

**ft_strchr.c** -> Locates the first occurence of a char in a string.

**ft_strdup.c** -> Allocates and returns a duplicate of a string.

**ft_striteri.c** -> Applies a function to each char of a string (in place) -> return the same string with the changes.

**ft_strjoin.c** -> Concatenates two string into a new allocated string.

**ft_strlcat.c** -> Concatenates two strings with size limitation.

**ft_strlcpy.c** -> copies a string into another buffer with size limitation.

**ft_strlen.c** -> Returns the length of a string.

**ft_strmapi.c** -> Applies a fonction to each char of a string and creates a new allocated string.

**ft_strncmp.c** -> Compares two strings up to a specified number of char.

**ft_strnstr.c** -> Locates a substring within a string, limited to a given length and return a pointeur to the first char of the substring.

**ft_strrchr.c** -> Locates the last occurence of a char in a string.

**ft_strtrim.c** -> Removes specified char from the beggining and end of a string -> return an allocated string without the char.

**ft_substr.c** -> Extract a substring from a sting -> return that allocated substring.

**ft_toupper.c** -> convert a lowercase letter to it's uppercase.

**ft_tolower.c** -> convert a uppercase letter to it's lowercase.
